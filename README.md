# Bird New Distribution Records

This repository stores bird new-record analyses as task-based reproducible folders.

## Structure
- `tasks/`: independent analytical tasks
- `source_data/`: cleaned source datasets used by the workflows
- `shared_assets/`: reusable assets such as bird silhouettes/icons
