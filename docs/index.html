Overview

This repository stores bird new-record analyses as task-based reproducible folders. Each task keeps code, derived data, figures, methods, and concise summaries together, making the project easier to browse and reuse across devices.

  • Source data: cleaned bird new-record datasets and species-trait tables.
  • Task modules: standalone workflows for directional analysis, temporal trends, spatial hotspots, and manuscript-style figures.
  • Outputs: publication-ready PNG, PDF, and PPTX figures plus reusable tables.
