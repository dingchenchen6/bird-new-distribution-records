# Bird Sankey Order Province Year

## Overview | 任务概述
This task prepares the standardized flow table for bird new records and renders an English Sankey diagram showing transitions from order to province to year.
本任务整理鸟类新纪录的标准化流向数据，并绘制英文版 `Order -> Province -> Year` 桑基图。

## Folder Structure | 文件夹结构
- `figures`: Sankey outputs in publication-ready formats
- `code`: annotated R script for data preparation and plotting
- `data`: plot-ready CSV files and the four-field summary table
- `results`: concise task summary

## Main Script | 主脚本
- `code/make_bird_sankey_order_province_year_en.R`

## Key Outputs | 主要输出
- `figures/fig_ref01_sankey_order_province_year_en_v2.png`
- `data/bird_sankey_order_province_year_en_v2.csv`
- `data/鸟类新纪录_目-省份-年份-濒危等级.csv`
