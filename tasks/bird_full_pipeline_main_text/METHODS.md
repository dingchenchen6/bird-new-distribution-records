# Methods | 方法说明

## Objective | 目标
Run the integrated bird new-record analysis workflow for manuscript-scale outputs across temporal, spatial, taxonomic, and threat-status dimensions.
运行面向论文正文的一体化鸟类新纪录分析流程，覆盖时间、空间、分类与濒危等级多个维度。

## Workflow | 分析流程
1. Ingest and clean the bird record spreadsheet.
2. Generate derived summary tables for year, province, order, IUCN status, discovery reasons, and methods.
3. Build the full figure set for manuscript-scale interpretation.
4. Export all summary tables and main-text figures.

1. 读取并清洗鸟类新纪录原始表格。
2. 生成年份、省份、目、IUCN等级、发现原因和发现方式等汇总表。
3. 构建正文所需的完整图件集合。
4. 导出全部汇总表和正文图件。
