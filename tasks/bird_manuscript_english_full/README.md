# Bird manuscript task
# 鸟类新纪录英文论文任务

This task converts the completed bird new-record analyses into a full English manuscript package suitable for continued journal drafting.
本任务将已完成的鸟类新纪录分析整合为一套可继续修改和投稿准备的英文论文稿件包。

## Contents / 内容
- `code`: figure-building and DOCX-building scripts.
- `data`: figure plan and manuscript build diagnostics.
- `figures`: manuscript-ready main figures and one supplementary figure.
- `results`: markdown source and final Word manuscript.

## Main deliverables / 核心产出
- Full English manuscript in Word format.
- Manuscript markdown source for version control.
- Combined main figures used in the manuscript.
- Diagnostics confirming figure/table/image embedding.
