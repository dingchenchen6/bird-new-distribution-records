# Bird Binomial Order Pattern

## Overview | 任务概述
This task recreates the manuscript-style panel `(a)` using a binomial expectation envelope and observed order-level proportions for bird new-distribution records.
本任务使用二项分布期望包络线与观测值，复刻论文风格的 `(a)` 面板并展示鸟类新纪录在目层面的偏离模式。

## Folder Structure | 文件夹结构
- `figures`: final figure outputs
- `code`: annotated R script
- `data`: plot data and the binomial envelope table
- `results`: concise task summary

## Main Script | 主脚本
- `code/make_bird_fig_a_binomial_order_pattern.R`

## Key Outputs | 主要输出
- `figures/fig_ref04_order_binomial_pattern_a.png`
- `data/bird_fig_a_order_binomial_plot_data.csv`
- `data/bird_fig_a_order_binomial_envelope.csv`
