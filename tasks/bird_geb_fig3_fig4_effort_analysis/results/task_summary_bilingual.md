# Bird GEB-style Figure 3 and Figure 4 analysis / 鸟类 GEB 风格 Figure 3 与 Figure 4 分析摘要

## English
- Species-level models were fitted for 1258 bird species with sufficiently complete trait data.
- Range size was positively associated with whether a species yielded any new provincial record (beta = 0.255, 95% CI 0.131 to 0.378).
- Range size also showed a positive association with the number of new provincial records in the count model (beta = 0.229, 95% CI 0.132 to 0.326).
- Province-level analysis included 31 provinces with complete structural covariates. After area standardization, current survey effort showed a positive relationship with discovery intensity (beta = 0.449, p = 0.0304), whereas province area showed a negative relationship (beta = -0.589, p = 7.15e-05).
- The primary province model explained 71.1% of the variance in log-transformed discovery intensity, and hierarchical partitioning identified current survey effort and province area as the largest contributors.
- High-collinearity screening showed that current report and current user effort were nearly redundant (joint VIF > 384.7), so they were not entered together in the same primary model.
- Influence screening identified the following provinces above the 4/n Cook's-distance threshold: Jiangsu, Tianjin. After removing these provinces, the effect directions for current survey effort (beta = 0.607) and area (beta = -0.591) remained unchanged.
- Replacing current report effort with current user effort yielded a positive user-effort coefficient (beta = 0.492), indicating that the effort effect is qualitatively robust to proxy choice.

## 中文
- 物种层模型共纳入 1258 个性状信息较完整的鸟种。
- 分布范围对物种是否产生新省级纪录具有正向作用（beta = 0.255，95% CI 0.131 至 0.378）。
- 在计数模型中，分布范围对新省级纪录数量同样表现出正向影响（beta = 0.229，95% CI 0.132 至 0.326）。
- 省级分析共纳入 31 个具有完整结构协变量的省级单元。面积标准化后，当前调查努力与新纪录发现强度呈显著正相关（beta = 0.449，p = 0.0304），而省域面积呈显著负相关（beta = -0.589，p = 7.15e-05）。
- 省级主模型解释了对数转换后发现强度变异的 71.1%；层次分解结果表明，当前调查努力和省域面积是贡献最大的两个解释变量。
- 共线性筛查表明，当前报告数努力与当前用户数努力几乎冗余（联合 VIF > 384.7），因此主模型中不同时纳入两者。
- 影响点筛查识别出 Cook's distance 高于 4/n 阈值的省份为：Jiangsu, Tianjin。剔除这些省份后，当前调查努力（beta = 0.607）和面积（beta = -0.591）的作用方向保持不变。
- 将当前报告数努力替换为当前用户数努力后，用户努力系数仍为正（beta = 0.492），说明努力效应对代理指标的选择具有定性稳健性。
