# Bird Reference Style Figures

## Overview | 任务概述
This task recreates the major reference-style figures for bird new-distribution records, including map, Sankey, chord, phylogeny-style, and directional radar panels.
本任务复刻鸟类新纪录的参考风格图件，包括地图、桑基图、弦图、系统发育风格图和方向雷达图。

## Folder Structure | 文件夹结构
- `figures`: main reference-style figures
- `code`: annotated R script for the figure bundle
- `data`: cleaned input tables and figure-specific plot tables
- `results`: concise task summary

## Main Script | 主脚本
- `code/bird_reference_style_figures.R`

## Key Outputs | 主要输出
- `figures/fig_ref00_china_projection_province_records.png`
- `figures/fig_ref01_sankey_order_province_year.png`
- `figures/fig_ref01b_chord_order_province.png`
- `figures/fig_ref02_phylogeny_style_bird_new_records.png`
- `figures/fig_ref03_direction_radar_bird_new_records.png`
