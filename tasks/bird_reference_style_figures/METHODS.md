# Methods | 方法说明

## Objective | 目标
Produce a coherent set of manuscript-style reference figures for bird new-distribution records, following the layout logic and color language of the related mammal study.
参照相关哺乳动物研究的版式逻辑和配色语言，为鸟类新纪录输出一组风格统一的论文图件。

## Workflow | 分析流程
1. Clean the main bird record table and standardize taxonomic, spatial, temporal, and threat-status fields.
2. Build figure-specific tables for map, Sankey, chord, phylogeny-style, and directional panels.
3. Apply consistent typography, color mapping, and export settings.
4. Export figures and the bundled figure-data workbook.

1. 清洗主鸟类新纪录表，并标准化分类、空间、时间和濒危等级字段。
2. 为地图、桑基图、弦图、系统发育风格图和方向图分别构建作图数据。
3. 统一字体、配色和导出参数。
4. 导出图件与图表数据总表。
