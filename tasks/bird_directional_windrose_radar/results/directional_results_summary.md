# Bird directional plots: result summary
# 鸟类新纪录方向图：结果摘要

## Overview / 总体概况
- Total bird new-record species used in directional analysis: 648.
- 纳入方向分析的鸟类新纪录物种总数：648。
- Main 4x4 figures include the top 16 orders ranked by species counts.
- 4x4 主图纳入新纪录物种数排名前 16 的鸟类目。

## Overall dominant directions / 总体优势方向
- Top overall directions: East (234, 36.1%); Northeast (138, 21.3%); Southeast (72, 11.1%).
- 总体上占比最高的方向为：East（234，36.1%）；Northeast（138，21.3%）；Southeast（72，11.1%）。

## Included and omitted orders / 纳入与舍弃的目
- Included orders: Passeriformes, Charadriiformes, Accipitriformes, Anseriformes, Pelecaniformes, Columbiformes, Gruiformes, Coraciiformes, Strigiformes, Galliformes, Cuculiformes, Piciformes, Gaviiformes, Ciconiiformes, Procellariiformes, Suliformes.
- Dropped small-count orders: Caprimulgiformes, Podicipediformes, Falconiformes, Phoenicopteriformes, Otidiformes, Pterocliformes, Trogoniformes.
- 纳入的目：Passeriformes、Charadriiformes、Accipitriformes、Anseriformes、Pelecaniformes、Columbiformes、Gruiformes、Coraciiformes、Strigiformes、Galliformes、Cuculiformes、Piciformes、Gaviiformes、Ciconiiformes、Procellariiformes、Suliformes。
- 舍弃的小样本目：Caprimulgiformes、Podicipediformes、Falconiformes、Phoenicopteriformes、Otidiformes、Pterocliformes、Trogoniformes。

## Output structure / 输出结构
- `figures/overall`: main 4x4 order-panel figures used as the primary directional summary.
- `figures/combined`: top-order overlay comparison figures.
- `figures/radar_by_order` and `figures/windrose_by_order`: individual order-specific plots.
- `figures/overall`：4x4 按目主图。
- `figures/combined`：主要目叠加对比图。
- `figures/radar_by_order` 和 `figures/windrose_by_order`：各目单独图。
