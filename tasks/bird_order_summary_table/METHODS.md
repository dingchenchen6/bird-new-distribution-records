# Methods | 方法说明

## Objective | 目标
Summarize bird new-distribution records by order and generate a standardized table suitable for appendices and supplementary materials.
按目汇总鸟类新纪录，并生成适合正文附表或补充材料的标准化统计表。

## Workflow | 分析流程
1. Count unique new-record species and associated papers for each order.
2. Join the full order-level species pool reference table.
3. Calculate the proportion of each order among all new-record species and the proportion of new-record species within the full order species pool.
4. Export display-ready and numeric tables, plus a rendered figure version.

1. 统计各目的新纪录物种数和对应文章数。
2. 连接目级物种总数参考表。
3. 计算“占全部新纪录物种比例”和“占该目物种总数比例”。
4. 导出展示版、数值版和表格图像版。
