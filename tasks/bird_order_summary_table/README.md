# Bird Order Summary Table

## Overview | 任务概述
This task produces a manuscript-style summary table by order, including new-record species counts, paper counts, and two proportional indicators.
本任务生成按目汇总的论文风格统计表，包括新纪录物种数、文章数以及两个比例指标。

## Folder Structure | 文件夹结构
- `figures`: rendered table preview
- `code`: annotated R script
- `data`: numeric table, display table, markdown table, and species pool reference table
- `results`: concise task summary

## Main Script | 主脚本
- `code/make_bird_order_summary_table.R`

## Key Outputs | 主要输出
- `figures/table_order_summary_bird_new_records.png`
- `data/table_order_summary_bird_new_records.xlsx`
- `data/table_order_summary_bird_new_records_display.csv`
